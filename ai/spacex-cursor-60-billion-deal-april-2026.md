# SpaceX just offered to buy Cursor for $60 billion. A code editor. $60 billion.

A 25-year-old CEO, a $2B fundraise that got preempted overnight, Microsoft watching from the sidelines, and Elon Musk's play to own the future of software engineering.

---

Let me say the number again because it doesn't feel real. $60 billion. For a code editor.

Cursor - the AI-powered code editor that launched in 2023, built by a team of 25-year-olds, and currently being used by basically every developer who's tried it - has agreed to let SpaceX buy it for $60 billion later this year. Or, if SpaceX doesn't pull the trigger, they'll pay $10 billion for the work they're doing together. Either way, Cursor's founders just became some of the richest people in tech.

## What actually happened

Cursor was in the middle of closing a $2 billion fundraise. Standard stuff for a hot AI company in 2026. Then SpaceX showed up, offered a $10 billion "collaboration fee" and a path to a $60 billion acquisition, and Cursor halted the fundraising talks.

The structure is unusual. SpaceX isn't buying Cursor today. They're buying the right to acquire it later this year - likely after SpaceX's IPO this summer, which is shaping up to be the biggest IPO in history. The timing is deliberate: SpaceX wants to avoid updating its confidential financial filings before the listing, and paying for Cursor with publicly traded stock is cheaper than paying cash.

## Why SpaceX wants a code editor

This only makes sense if you remember that SpaceX merged with xAI (Musk's AI lab) in February in a deal Musk valued at $1.25 trillion. So this isn't SpaceX the rocket company buying a code editor. It's xAI - which has the Colossus supercomputer with millions of H100 GPUs - acquiring the best AI coding product on the market.

The pitch from SpaceX's announcement: Cursor's "leading product and distribution to expert software engineers" combined with xAI's "million H100 equivalent Colossus training supercomputer" will allow them to "build the world's most useful models."

Translation: Cursor has the users and the UX. xAI has the compute. Together they can build coding models that OpenAI (Codex) and Anthropic (Claude Code) can't match.

## The numbers in context

- Cursor's revenue is reportedly $500M+ ARR
- That's a 120x revenue multiple at $60B
- For comparison, GitHub sold to Microsoft for $7.5B in 2018
- Cursor is worth 8x what GitHub was worth
- The CEO, Michael Truell, is 25 years old and a former Google intern

## Microsoft was looking too

CNBC reported that Microsoft explored buying Cursor before SpaceX made its move. Microsoft, which owns GitHub and has its own AI coding tool (GitHub Copilot powered by OpenAI), looked at Cursor and passed. Whether they passed because the price was too high or because they thought Copilot was enough, the fact that they were looking tells you everything about how seriously the industry takes Cursor as a threat.

## My take

This deal is either the most important acquisition in AI history or the most overpriced. I genuinely can't tell which.

On one hand, $60 billion for a company with $500M in revenue is absurd by any traditional metric. The 120x multiple makes the dot-com era look conservative. If AI coding tools get commoditized - and they might, given that Claude Code, Copilot, and a dozen others are all getting better fast - then Cursor at $60B will look like Yahoo overpaying for Tumblr, except 1000x worse.

On the other hand, if AI-assisted coding becomes the default way all software is written (which it will), then owning the dominant interface for that transition is worth nearly anything. Cursor isn't just a code editor - it's the IDE that an entire generation of developers is switching to. That kind of distribution is genuinely rare.

The SpaceX/xAI angle makes it more interesting. If Musk can combine Cursor's UX with xAI's compute to train coding models that are meaningfully better than what OpenAI and Anthropic offer, then $60B might look cheap in hindsight. The entire software industry runs through code editors. Owning the best one, with the best AI, is owning the bottleneck.

But $60 billion. For a company founded three years ago. By a 25-year-old. That doesn't stop being wild no matter how you frame it.
